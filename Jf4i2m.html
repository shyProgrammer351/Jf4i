문제

Autoboxing과 Unboxing의 예제 문제입니다 이를 해결하는 다음 프로그램의 실행상태를 확인하세요 
num1 = 20, num2 = 20

프로그램 코드

		public class AutoboxingUnboxingExample 
	{
M1b		public static void main( String[] args ) 
		{
M11			int num1 = 10;
	
M12			Integer num2 = num1;
M13			num2 = Integer.parseInt( "20" );
	
M14			num1 = num2;
			System.out.println( "num1 = " + num1 + ", num2 = " + num2);
M1e		}
	}
