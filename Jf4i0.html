비교 연산자

기호 명칭 참(true) 예제 거짓(false) 예제
== 등가연산자 같은 값 == 같은 값 5 == 5 다른 값1 == 다른 값2 3 == 7

compareTo   클래스 객체의 데이터 값과 전달인자의 데이터 값을 비교한 차이를 반환하며, 같으면 0을 반환

equals   클래스 객체의 데이터 값과 전달인자의 데이터 값을 비교하여 같으면 true, 다르면 false를 반환
 
    var1   (Integer객체01)       (Integer객체01)   10   등의 데이터와 메서드 포함
var2 (Integer객체02)   (Integer객체02)   10   등의 데이터와 메서드 포함

※ 주의사항
var1.equals( var2 ) : 변수에서 참조하는 객체에 저장된 값이 같으면 true, 10==10은 true
var1 == var2 : 변수값이 같으면 true, (Integer객체01)==(Integer객체02)는 false
var1 = var2 : 왼쪽 변수값을 오른쪽 변수공간에 저장, var1=(Integer객체02) 대입
