A양이 가위바위보를 선택하면 B군이 임의로 가위바위보를 선택하고 지는 가위바위보의 승패를 출력하는 문제입니다 실행순서를 클릭하세요 
가위, 바위, 보 중 하나를 입력하세요 : 가위
플레이어 A는 가위를 냈습니다.
플레이어 B는 보를 냈습니다.
판정결과는 플레이어 B가 이겼습니다

			int num = (int)(Math.random()*3);

			String result = "";

			}

			}

					|| ( playerA.equals( "보" ) && playerB.equals( "바위" ) ) )

			}

			else

			{

				result = "플레이어 B가 이겼습니다";
